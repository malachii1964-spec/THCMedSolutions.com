# Palette Library
# Starting points, not law — design.md process still applies (name 4-6 hexes
# per project, derived from the SUBJECT). Never default to any of these blind.

EARTHBOUND WARMTH — calm, grounded, eco.
F6F3EB (cloud white) / C9CAAC (dry sage) / 869B7E (moss) / 7F2020 (oxblood accent)
Best: wellness, sustainable, food. Note: dangerously close to the cream-serif
AI tell — pair with unexpected type to stay distinctive.

LUMINOUS NEON / CYBER — futurist, Y2K, energetic.
FF61F8 (cyber pink) / FFA6FB (soft pink glow) / FFFBA7 (acid butter) / FFEA6C (voltage yellow)
Best: gaming, youth, music. Use on dark or shocking-light base; 1-2 neons max at once.

COOL BLUE x WASABI — bold optimism, fashion-forward.
A7C7E7 (frost blue) / 2E5E4E (jade) / 2B1B2C (plum noir) / B7D336 (wasabi) / F45D48 (persimmon)
Best: fashion, beauty, creative portfolios. Wasabi/persimmon are accents, not fields.

AMBIENT GLOW — cinematic dark with radiant color light.
0B0D12 (near-black base) / glow gradients: 4C6FFF -> 9B4CFF or FF6B4C -> FFB84C
Best: AI SaaS, dashboards, launch pages. Glow behind content, never on text.
