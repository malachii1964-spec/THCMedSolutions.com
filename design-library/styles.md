# Style Directions Library
# Interview: when the human has no reference sites, offer 2-3 of these BY NAME
# based on the subject. Each entry: vibe / best for / signature techniques / trap.

MINIMALISM — negative space, few elements, typographic confidence.
Best: premium tech, portfolios, SaaS. Techniques: oversized type, 1 accent,
whitespace as structure. Trap: minimal ≠ empty; demands perfect spacing.

MAXIMALISM — dense color, stacked patterns, controlled chaos.
Best: music, festivals, creative brands. Techniques: layered collage, marquee
text, clashing-on-purpose palette. Trap: needs a grid underneath or it's noise.

EDITORIAL — magazine DNA: strong grid, serif display, pull quotes, captions.
Best: blogs, journalism, agencies. Techniques: multi-column grids, drop caps,
image captions as design. Trap: don't fake it with hairlines everywhere (banned tell).

BRUTALISM / NEO-BRUTALISM — raw blocks, hard borders, unpolished honesty.
Best: dev tools, streetwear, anti-corporate brands. Techniques: thick outlines,
hard shadows, system-font energy, visible structure. Trap: still needs hierarchy.

Y2K / CYBER — chrome, neon, gradients, retro-future nostalgia.
Best: gaming, youth brands, music. Techniques: metallic text, glow, scanlines,
cursor effects. Trap: heaviest slop-risk of the set; commit fully or skip.

ORGANIC / EARTHBOUND — nature materials, soft shapes, calm.
Best: wellness, sustainable brands, food. Techniques: grain/paper texture,
blob-free soft radii, botanical photography. Trap: sage-green sameness; add one
unexpected element.

GLASS / LUMINOUS — depth via blur, ambient glow, cinematic dark scenes.
Best: AI/fintech SaaS, dashboards. Techniques: glassmorphism panels (sparingly),
glow gradients behind content, dark base. Trap: blur is a GPU tax — budget it.

IMMERSIVE 3D — see immersive-3d.md. The showstopper with the sharpest
performance teeth.

RETRO PRINT — risograph/zine energy: limited inks, halftones, off-registration.
Best: events, coffee, indie products. Techniques: 2-3 ink colors, halftone
textures, stamp/sticker elements. Trap: readability first, texture second.
